How it works

The Intelligent Dispatch Engine (IDE) evaluates every available driver-truck combination based on cargo weight, truck capacity, driver performance, and real-time proximity.

  • Auto-Assignment: Triggered if a score exceeds the threshold.
  • Manual Fallback: If no candidate meets the threshold, the system flags the shipment for review.
  • Audit Trail: Every decision is logged for operational analysis.
